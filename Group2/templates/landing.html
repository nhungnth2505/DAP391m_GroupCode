System Architecture

ML Pipeline + LLM Advisor

The system processes student survey data through a multi-stage machine learning pipeline, then feeds prediction results as context to an LLM-powered chatbot for personalized guidance.

📋
Stage 1
Survey Input
20–40 features across psychological, academic, environmental & social dimensions
⚙️
Stage 2
Preprocessing
Normalization, encoding, feature selection (SelectKBest, RFECV), PCA reduction
🧠
Stage 3
ML Models
9 classifiers: SVM, CatBoost, LightGBM, XGBoost, Random Forest, Gradient Boosting, AdaBoost, Bagging, Decision Tree
📊
Stage 4
Prediction Output
Career path + RIASEC profile · Final grade G3 · Stress level with confidence scores
🔗
Stage 5
Context Builder
Structured profile: career → performance → stress history assembled as LLM system context
💬
Stage 6
LLM Advisor
Context-aware chatbot answers career, academic & stress questions only
🎯
Multi-Model Ensemble 9 classifier models evaluated per prediction task with hyperparameter tuning via GridSearchCV
🔬
Feature Engineering SelectKBest, RFECV, PCA — multiple preprocessing configurations benchmarked per model
💬
Context-Aware LLM Prediction results injected as system context — strictly answers career, performance & stress topics only
